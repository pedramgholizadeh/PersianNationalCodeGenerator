
# Persian National Code Generator

a simple chrome extention!
- Generate fake national code 
- Check validation

# Usage

follow these steps :

- clone repository
- unzip as folder
- go to ``chrome://extensions/``
- use ``Load unpacked``
- select folder
- ENJOY
